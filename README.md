# Stock Market Prediction 

A comparative machine learning project that evaluates traditional regression algorithms against deep learning architectures for time-series stock price forecasting.

## Models Implemented
This repository contains individual implementations to compare how different architectures handle market volatility:
* **Linear Regression (`Linear_Regression.ipynb`):** Establishes a baseline predictive model.
* **Support Vector Regression (`SVR.ipynb`):** Explores non-linear mappings using the RBF kernel.
* **Random Forest Regressor (`Random_Forest.ipynb`):** An ensemble tree-based method to capture complex feature interactions.
* **Long Short-Term Memory (`LSTM.ipynb`):** A Recurrent Neural Network (RNN) designed specifically to retain long-term sequential dependencies in time-series data.

## Tech Stack & Libraries
- **Language:** Python
- **Libraries:** NumPy, Pandas, Scikit-Learn, TensorFlow/Keras (for LSTM), Matplotlib, Seaborn
